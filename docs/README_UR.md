**آواز سے مطلب، نہ کہ آواز سے لفظ۔**

اپنی زبان میں بولیں۔ کسی بھی دوسری زبان میں متن حاصل کریں۔

macOS کے مینو بار کا ایک ٹول جو آپ کی آواز ریکارڈ کرتا ہے اور صاف، فطری متن آپ کی مطلوبہ زبان میں فراہم کرتا ہے۔ یہ لفظ بہ لفظ ترجمہ نہیں ہے۔ Murmur آپ کی بات سمجھتا ہے اور اسے ایسے لکھتا ہے جیسے کوئی مقامی بولنے والا لکھے۔

## 😤 مسئلہ

غیر مادری زبان میں لکھنا سست عمل ہے۔ آپ یا تو:

- اپنی زبان میں لکھتے ہیں، ترجمے میں پیسٹ کرتے ہیں، پھر عجیب نتیجے کو درست کرتے ہیں
- سیدھا مطلوبہ زبان میں لکھتے ہیں، ہر لفظ پر شک کرتے ہیں، تلاش کرتے ہیں، بار بار پڑھتے ہیں کہ صحیح لگ رہا ہے یا نہیں
- AI سے ترجمہ کرواتے ہیں، پھر ایڈٹنگ میں وقت لگاتے ہیں کیونکہ نتیجہ بہت لفظی یا لہجے سے ہٹا ہوا ہوتا ہے

یہ مسئلہ اور بھی بڑھ جاتا ہے جب آپ AI ایجنٹس کے ساتھ کام کرتے ہیں، جہاں انگریزی بہتر انتخاب ہے (کم ٹوکنز، ماڈل کی بہتر سمجھ)، لیکن سوچنے کا عمل آپ کی مادری زبان میں ہوتا ہے۔

## 💡 حل

ہاٹ کی دبائیں۔ جو کہنا ہے کہیں۔ مطلوبہ زبان میں تیار پیغام حاصل کریں۔

```
Option+Space  →  کسی بھی زبان میں بولیں  →  Option+Space
                                               ↓
                                 صاف متن وہیں آ جاتا ہے جہاں آپ ٹائپ کر رہے ہیں
```

Murmur لفظ بہ لفظ ترجمہ نہیں کرتا۔ یہ آپ کی بولی ہوئی بات لیتا ہے، فالتو الفاظ اور زبانی شور ہٹاتا ہے، اور ایسا متن تیار کرتا ہے جو مطلوبہ زبان کی گرامر، لہجے اور روایات کے مطابق ہو۔ نتیجہ ایسا لگتا ہے جیسے لکھا گیا ہو، ترجمہ نہیں۔

## ⚙️ یہ کیسے کام کرتا ہے

1. **ہاٹ کی دبائیں** (ڈیفالٹ `Option + Space`)۔ ریکارڈنگ انڈیکیٹر ظاہر ہوگا۔
2. **بولیں** کسی بھی زبان میں۔ جیسے سوچتے ہیں ویسے کہیں۔
3. **کلک کریں** کسی بھی ٹیکسٹ فیلڈ میں (براؤزر، ایڈیٹر، میسنجر، ٹرمینل)۔
4. **دوبارہ ہاٹ کی دبائیں**۔ متن وہیں آ جائے گا جہاں آپ کو چاہیے۔

ایپ سوئچنگ نہیں۔ کاپی پیسٹ نہیں۔ ایڈٹنگ نہیں۔

## 🔀 تین موڈز

- **Transcription**: بولی ہوئی زبان میں خام تقریر سے متن۔
- **Clean-up**: وہی زبان، لیکن صاف شدہ۔ فالتو الفاظ نہیں، درست گرامر، مربوط جملے۔ فہرستیں خود بخود فارمیٹ ہو جاتی ہیں۔
- **Translation**: ایک زبان میں بولیں، دوسری زبان میں صاف متن حاصل کریں۔ 97 زبانیں سپورٹ ہیں۔ وہی صفائی لاگو ہوتی ہے: نتیجہ ایسا لگتا ہے جیسے مقامی بولنے والے نے لکھا ہو، ترجمہ نہیں۔

## 📦 انسٹال

[Releases](https://github.com/alexe-ev/Murmur/releases) سے `Murmur.dmg` ڈاؤن لوڈ کریں، Applications میں ڈریگ کریں۔

> نوٹرائزڈ نہیں ہے۔ پہلی بار: ایپ پر رائٹ کلک کریں، پھر Open دبائیں۔

### سورس سے بلڈ

```bash
git clone https://github.com/alexe-ev/Murmur.git
cd Murmur
xcodebuild -scheme Murmur -configuration Release -derivedDataPath build
cp -R build/Build/Products/Release/Murmur.app /Applications/
```

### سیٹ اپ

پہلی بار چلانے پر **Microphone** اور **Accessibility** کی اجازتیں دیں۔ Settings میں اپنی [OpenAI API key](https://platform.openai.com/api-keys) درج کریں۔

## 📋 ضروریات

- macOS 13.0+ (Ventura یا بعد کا)
- Apple Silicon (M1/M2/M3/M4)
- OpenAI API key

آڈیو ریکارڈنگز عارضی ہیں اور ٹرانسکرپشن کے فوراً بعد حذف ہو جاتی ہیں۔ آپ کی API key آپ کی مشین پر مقامی طور پر محفوظ ہوتی ہے۔
